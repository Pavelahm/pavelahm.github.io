1. Linear Regression Overview

Linear regression models the relationship between a dependent variable and one or more independent variables. The model is written as:

Y = β₀ + β₁X + ε

  • Y = dependent variable
  • X = independent variable
  • β₀ = intercept
  • β₁ = slope
  • ε = error term

Key Concepts

  • Simple Linear Regression
  • Multiple Linear Regression
  • Least Squares Estimation
  • Model Evaluation (R², etc.)

Discussion

Graphs should be placed after the model explanation section to support interpretation. The dependent variable is the outcome being predicted, such as performance or behavior.
